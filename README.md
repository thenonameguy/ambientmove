ambientmove
===========

Ambient lighting with the PSMove (API)

===========
<h2>Install</h2>

Clone it, then simply

    make

===========
<h2>Requires:</h2>
- 1 Playstation Move controller
- <a href="https://github.com/thp/psmoveapi">PSMove API</a>

Props to <a href="http://thp.io/">Thomas Perl</a> for his AWESOME API!
